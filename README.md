#🎵 WaveTune-Music-Player
A simple and responsive web-based music player built using HTML, CSS, and JavaScript.
This project allows users to play, pause, and navigate between songs with a clean UI and interactive controls.

🚀 Features
▶️ Play / Pause music
⏭️ Next & Previous song navigation
📊 Interactive progress bar
⏱️ Real-time song duration & current time
🖼️ Dynamic song image display
📱 Responsive design using Bootstrap

🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Bootstrap 5
Bootstrap Icons

📂 Project Structure
📁 Music-Player
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 script.js
 ┣ 📄 data.js
 ┗ 📁 songs

 ⚙️ How to Run
Download or clone this repository
Make sure all song files are inside the songs folder
Open index.html in your browser
